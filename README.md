# chip8

## chip8 components:
- 16 8 bit registers
- 4096 bytes of memory
- 16 bit index register
- 16 bit program counter
- 16 level stack
- 8 bit stack pointer
- 8 bit delay timer
- 8 bit sound timer
- 16 input keys
- 64 x 32 monochrome display memory
